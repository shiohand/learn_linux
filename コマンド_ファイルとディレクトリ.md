## ファイルとディレクトリ

### pwd print-working-directory
`pwd`
カレントディレクトリの絶対パスを表示する

### cd change-directory
`cd ディレクトリ`
ディレクトリを移動する

### ls list
`ls [filter]`
ディレクトリとファイルの一覧表示

| option | long    | 動作                   |
| ------ | ------- | ---------------------- |
| -a     | all     | 隠しファイルも表示     |
| -f     | -       | ファイル種別を表示する |
| -l     | long    | 長い形式で表示         |
| -t     | time    | 更新時間でソート       |
| -r     | reverse | 逆順にソート           |

#### -fオプション
ファイル名+記号 としてファイル種別を表示する

| 表示 | 種別               |
| ---- | ------------------ |
| `/`  | ディレクトリ       |
| `@`  | シンボリックリンク |
| `*`  | 実行可能ファイル   |
| なし | 通常ファイル       |

### find find
`find パス 検索文字列 [検索アクション]`
指定パス配下のファイルを検索する

ファイル名で検索
`find パス -name ファイル名 [検索アクション]`
カレント以下の`-type d`の`-name images`を検索して`-print`
`find . -type d -name images -print`

| option     | long | 動作                                   |
| ---------- | ---- | -------------------------------------- |
| -i         |      | ファイル名で検索                       |
| -iname     |      | 大文字小文字を区別しない               |
| -type [fd] |      | fでファイル、dでディレクトリに絞り込み |

### cp copy
`cp コピー元 コピー先`
`cp コピー元ファイル... コピー先ディレクトリ`
コピーを作成する
上書き 別ディレクトリへの同名コピーも可能

ディレクトリをコピーしたい場合は`-r`オプションが必要

| コピー先       | 動作                               |
| -------------- | ---------------------------------- |
| 既存のファイル | 上書き                             |
| ディレクトリ   | そのディレクトリの中に同名でコピー |

| option | long     | 動作                       |
| ------ | -------- | -------------------------- |
| -i     |          |                            |
| -f     |          |                            |
| -r     |          |                            |
| -p     | property | 元ファイルの情報を保持する |
| -P     | parents  |                            |

### mv move
`mv 移動元 移動先`
`mv 移動元ファイル... 移動先ディレクトリ`
ファイルやディレクトリを移動する

移動先をファイルにすると名前の変更になる

| option | long | 動作 |
| ------ | ---- | ---- |
| -i     |      |      |
| -f     |      |      |

### rm remove
`rm 削除対象...`
ファイルを削除する
`rmdir`では削除できない空でないディレクトリも`-r`で削除できる

| option | long | 動作 |
| ------ | ---- | ---- |
| -i     |      |      |
| -f     |      |      |
| -r     |      |      |

## ディレクトリ

### mkdir make-directory
`mkdir ディレクトリ...`
ディレクトリを作成する
階層のあるディレクトリの作成には`-p`が必要

| option | long    | 動作                         |
| ------ | ------- | ---------------------------- |
| -p     | parents | 階層のあるディレクトリの作成 |

#### rmdir remove-directory
`rmdir ディレクトリ...`
ディレクトリが空である場合に削除する
空でない場合は削除できない

## ファイル

### touch
`touch ファイル...`
新規ファイルを作成する
本当はファイルのタイムスタンプを更新するコマンド

### cat concatenate
`cat ファイル...`
ファイルの内容を表示する

| option | long          | 動作                    |
| ------ | ------------- | ----------------------- |
| -n     | number        | 行番号を表示            |
| -s     | squeeze-blank | 連続した空行を1行にする |

#### cat で直接入力した内容でファイルを作成する
`cat > [ファイル]`

入力画面が表示され文章を入力できる。Ctrl + [D]で終了する。
入力した内容をリダイレクトでファイルに記述する(-> 標準入出力.md)

#### more, less
`more ファイル...`
`less ファイル...`
ページングされる

| 共通の操作 | 動作                       |
| ---------- | -------------------------- |
| スペース   | 次のページへ               |
| b          | 一画面前に戻る             |
| f          | 一画面次に進む             |
| /単語      | 単語を後方検索 nで次 Nで前 |
| ?単語      | 単語を前方検索             |
| q          | 終了                       |

| lessの操作 | 動作     |
| ---------- | -------- |
| 上矢印     | 一行前へ |
| 下矢印     | 一行次へ |

### head
`head ファイル...`
先頭から10行まで表示する

| option | long      | 動作                        |
| ------ | --------- | --------------------------- |
| -n [N] | lines=[N] | 先頭からN行まで表示する     |
| -c [N] | bytes=[N] | 先頭からNバイトまで表示する |

### tail
`tail ファイル...`
末尾から10行まで表示する

ファイルへの変更をリアルタイムに反映する`-f`オプションは、ログファイルのモニタなどに用いられる

| option | long      | 動作                           |
| ------ | --------- | ------------------------------ |
| -n [N] | lines=[N] | 末尾からN行まで表示する        |
| -c [N] | bytes=[N] | 末尾からNバイトまで表示する    |
| -f     | follow    | 表示がリアルタイムに更新される |

### sort
`sort ファイル`
テキストファイルの中身をソートする

| option      | 動作                                               |
| ----------- | -------------------------------------------------- |
| -m          | ソート済みの複数ファイルを一つにソートする         |
| -r          | 逆順にソートする                                   |
| -k [n[, n]] | n列目の値でソートする                              |
| -b          | 先頭の空白を無視する                               |
| -f          | 小文字を大文字と同じものとみなしてソートする       |
| -n          | 数値としてソートする 先頭の空白は無視する          |
| -M          | 3文字の月表記としてソートする 先頭の空白は無視する |

#### uniq unique
`uniq ファイル`
同じ内容の行が連続しているときに一つにまとめる
連続していない場合は何もしない

### tr translate
`tr 文字列1 文字列2`
文字列1の各文字を対応する文字列2の各文字に置き換える

(例)`textfile`の内容の`a`を`A`, `b`を`B`, `c`を`C`に置き換えて`textfile2`に保存する
`$ cat textfile | tr abc ABC > textfile2`

### diff difference
`diff ファイル1 ファイル2`
ファイルの差分を表示する

| option | long    | 動作              |
| ------ | ------- | ----------------- |
| -c     | context | context形式で出力 |
| -u     | unified | unified形式で出力 |

## リンクを張る

- ハードリンク
エイリアスのような
元ファイルが消えてもハードリンクが参照している限り残っている
Javaのガベージコレクトをイメージ

- シンボリックリンク
Winの通常のlnkのような
元ファイルが移動・削除されるとリンク切れになる
用途として、`java`というリンクを常に最新バージョンを参照させるようにすることで、普段の利用が容易になる

### ln link
`ln 元ファイル 作成ファイル`

| option | long | 動作                     |
| ------ | ---- | ------------------------ |
| -s     |      | シンボリックリンクにする |
